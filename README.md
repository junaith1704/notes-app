# Flask Notes App

A simple web application to create, view, and delete notes using Flask and MongoDB Atlas.

## Features
- Add notes
- View all notes
- Delete notes

## Tech Stack
- Python (Flask)
- MongoDB Atlas
- HTML/CSS

## How to Run

1. Clone the repository:
   git clone https://github.com/junaith1704/notes-app.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   python app.py

## Project Structure
- app.py → main Flask application
- templates/ → HTML files
- requirements.txt → dependencies
